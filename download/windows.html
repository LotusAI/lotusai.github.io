How to install in Windows

There's a few steps to follow when installing Lotus.ai

  1. If you you not already done so, first reset your password for the email you originally signed up with or were provided.
  2. Download the current version of teh software agent using the button above.
  3. Unzip (extract) the Lotus AI - Windows Agent - x.x.x.zip file to a folder on your PC. It will contain 2 files: setup.exe (“Application”) and setup.msi (“Windows Installer Package”).
  4. Run the setup.exe file (not the .msi file).
  5. Follow the installation prompts to complete the installation.
  6. A shortcut with the following icon will appear on your desktop and in your Start menu.
  7. Run the application.

NOTE: During installation, a shortcut is created in your start-up folder. Each time you power on your computer the application will automatically start.

First Use

The first time you run the Agent you will need to login before it can start monitoring.

Enter your registered email and password.
After successful login you will see the following screen with your name.

There’s nothing more you need to do. The application will remember your login and automatically start with your profile each time you start your computer. You can now use the “minimise” or “close” icons (top right corner) to minimise the application to your System Tray, as seen here:

Double-click the icon to show the main window again. To exit the application and stop all monitoring, right click on the icon, and select the Exit option. It may take a moment to close as it finishes synchronising with the Lotus AI server.
